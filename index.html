Cryptography

NIST-approved.
Quantum-proof.

Commes implements all three NIST Post-Quantum Cryptography standards — the only suite secure against both classical and quantum adversaries.

Key Encapsulation

ML-KEM

Module-Lattice-based Key Encapsulation — formerly CRYSTALS-Kyber. Replaces RSA and ECDH for key exchange. Immune to Shor's algorithm.

Standard FIPS 203
Security level Category 3 (AES-192)
Public key size 1,184 bytes
Ciphertext size 1,088 bytes
Digital Signatures

ML-DSA

Module-Lattice-based Digital Signature Algorithm — formerly CRYSTALS-Dilithium. Replaces ECDSA for signing. Lattice-hard by construction.

Standard FIPS 204
Security level Category 3 (AES-192)
Signature size 3,293 bytes
Public key size 1,952 bytes
